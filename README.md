# StringProcessor — Лабораторная работа по обработке строк

**Java-приложение** для выполнения **трёх задач по обработке строк**:

---

## Задачи

1. **Вставка подстроки после N-го вхождения**  
   → Вставляет новую подстроку **после N-го** вхождения заданной подстроки.

2. **Поиск самого короткого и самого длинного слова**  
   → Возвращает **все слова** минимальной и максимальной длины.

3. **Проверка HTML-цвета (#RRGGBB)**  
   → Проверяет, является ли строка **валидным HEX-цветом** (исключая `#000000` и `#ffffff`).

## Как запустить

1. Убедитесь, что установлена **jdk1.8.0_144**
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/StringProcessor.git
   ```
3. Перейдите в папку проекта и выполните:
   ```bash
   javac -d out -cp . src/ru/rsreu/savushkin0217/*.java src/ru/rsreu/savushkin0217/classes/*.java
   java -cp out ru.rsreu.savushkin0217.ClientRunner
   ```

## Пример ввода/вывода

```
Введите количество строк: 1
Введите строку 1: Привет мир мир
Введите старое слово: мир
Введите новое слово: [новое]
Введите номер вхождения: 2

Результат:
1. Привет мир [новое] мир
2. Привет
3. NO, { Привет мир мир } isn't HTML colors code!
```
